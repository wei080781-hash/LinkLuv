# LinkLuv 媒體處理與最佳化完整指南

本文件為 LinkLuv 專案媒體處理邏輯的「最終完整版」。此文件包含所有步驟、完整的程式碼檔案內容與故障排除指南。

## 1. 系統環境準備

### 1.1 安裝 PHP 套件
在終端機 (專案根目錄) 執行：
```bash
composer require intervention/image-laravel
composer require php-ffmpeg/php-ffmpeg
php artisan vendor:publish --provider="Intervention\Image\Laravel\ServiceProvider"
```

### 1.2 設定 FFmpeg 環境變數 (Windows)
1. 從 [gyan.dev](https://www.gyan.dev/ffmpeg/builds/) 下載 `ffmpeg-release-full.7z`。
2. 解壓縮至 `D:\ffmpeg`。
3. 搜尋「環境變數」，編輯系統 Path，新增：`D:\ffmpeg\bin`。
4. **重要**：關閉並重啟所有終端機視窗，執行 `ffmpeg -version` 確認顯示版本號。

### 1.3 啟用 GD 擴展
1. 終端機執行 `php --ini` 尋找 `Loaded Configuration File` 路徑。
2. 開啟該檔案，搜尋 `extension=gd`。
3. 移除前面的分號 (`;`)，改為 `extension=gd`。
4. 若無 `extension_dir` 設定，請在檔案中新增 `extension_dir = "ext"`。
5. **務必完全重啟 Apache (XAMPP)** 或伺服器。

## 2. 核心程式碼實作

### 2.1 影片背景轉碼任務 (`app/Jobs/CompressVideoJob.php`)
請使用此完整內容建立或覆蓋該檔案：

```php
<?php

namespace App\Jobs;

use App\Models\Message;
use Illuminate\Bus\Queueable;
use Illuminate\Contracts\Queue\ShouldQueue;
use Illuminate\Foundation\Bus\Dispatchable;
use Illuminate\Queue\InteractsWithQueue;
use Illuminate\Queue\SerializesModels;
use FFMpeg\FFMpeg;
use FFMpeg\Format\Video\X264;
use Illuminate\Support\Facades\Storage;

class CompressVideoJob implements ShouldQueue
{
    use Dispatchable, InteractsWithQueue, Queueable, SerializesModels;

    protected $message;

    public function __construct(Message $message)
    {
        $this->message = $message;
    }

    public function handle()
    {
        $originalPath = $this->message->video_path;
        $compressedPath = 'videos/' . time() . '_compressed.mp4';
        
        $ffmpeg = FFMpeg::create();
        $video = $ffmpeg->open(storage_path('app/public/' . $originalPath));
        
        $format = new X264('libx264', 'libmp3lame');
        $format->setAdditionalParameters(['-crf', '28']); 

        $video->save($format, storage_path('app/public/' . $compressedPath));
        
        $this->message->update(['video_path' => $compressedPath]);
        Storage::disk('public')->delete($originalPath);
    }
}
```

### 2.2 控制器處理 (`app/Http/Controllers/MessageController.php`)
請使用此完整內容覆蓋該檔案：

```php
<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;
use Illuminate\Support\Facades\Cache;
use Illuminate\Support\Facades\Storage;
use Illuminate\Support\Facades\DB;
use Intervention\Image\ImageManager;
use Intervention\Image\Drivers\Gd\Driver;
use App\Models\Message;
use App\Jobs\CompressVideoJob;

class MessageController extends Controller
{
    public function store(Request $request)
    {
        $validated = $request->validate([
            'content' => 'required|string|max:1000',
            'parent_id' => 'nullable|exists:messages,id',
            'media' => 'nullable|file|mimes:jpg,jpeg,png,gif,mp4,mov,ogg|max:51200',
        ]);

        $parentId = $request->input('parent_id');
        $mediaPath = null;
        $mediaType = null;

        if ($request->hasFile('media')) {
            $file = $request->file('media');
            $mime = $file->getMimeType();

            if (str_contains($mime, 'image')) {
                $mediaPath = $this->handleImageUpload($file);
                $mediaType = 'image';
            } elseif (str_contains($mime, 'video')) {
                $mediaPath = $file->store('videos', 'public');
                $mediaType = 'video';
            }
        }
        
        $message = Message::create([
            'user_id'    => auth()->id(),
            'content'    => $request->content,
            'parent_id'  => $parentId,
            'image_path' => ($mediaType === 'image') ? $mediaPath : null,
            'video_path' => ($mediaType === 'video') ? $mediaPath : null,
            'media_type' => $mediaType,
            'depth'      => $parentId ? Message::findOrFail($parentId)->depth + 1 : 0,
        ]);

        $paddedId = str_pad($message->id, 10, '0', STR_PAD_LEFT);
        $message->update([
            'path' => $parentId ? (Message::findOrFail($parentId)->path . '.' . $paddedId) : $paddedId,
            'thread_id' => $parentId ? Message::findOrFail($parentId)->thread_id : $message->id
        ]);

        if ($parentId) {
            $this->storeClosure($message->id, $parentId);
        } else {
            DB::table('message_closure')->insert(['ancestor' => $message->id, 'descendant' => $message->id]);
        }

        if ($mediaType === 'video') {
            CompressVideoJob::dispatch($message);
        }

        Cache::forget('global_messages_feed');
        return response()->json(['success' => true]);
    }

    private function handleImageUpload($file)
    {
        $filename = 'images/' . time() . '_' . $file->hashName();
        $manager = new ImageManager(new Driver());
        $img = $manager->read($file);

        if ($img->width() > 1200) {
            $img->scale(width: 1200);
            $encoded = $img->toJpeg(quality: 80);
        } else {
            $encoded = $img->toJpeg(quality: 90);
        }
        
        Storage::disk('public')->put($filename, (string) $encoded);
        return $filename;
    }

    private function storeClosure($descendantId, $parentId) 
    {
        DB::table('message_closure')->insert(['ancestor' => $descendantId, 'descendant' => $descendantId]);
        $ancestors = DB::table('message_closure')->where('descendant', $parentId)->get();
        foreach ($ancestors as $a) {
            DB::table('message_closure')->insert(['ancestor' => $a->ancestor, 'descendant' => $descendantId]);
        }
    }
}
```

## 3. 系統啟動與驗證
1.  **啟動 Queue 服務** (必要)：在終端機執行 `php artisan queue:work`。
2.  **啟動伺服器**：執行 `php artisan serve`。
3.  **驗證流程**：
    *   圖片上傳：檢查 `storage/app/public/images` 大小是否縮減。
    *   影片上傳：觀察 `queue:work` 終端機，確認 `CompressVideoJob` 是否處理完成。
